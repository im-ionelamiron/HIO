HIO TECHNOLOGY

HIO is a set of Artificial Intelligence and Machine Learning algorithms able to simulate some Humans Cognitive Processes able to managing unexpected behaviors.

HIO algorithms have been created studying the cognitive processes of humans in phase of language learning, dialogues management, ideas invention, applying of logic deduction.

Thanks to our studies, we have understood how to create a General AI without using the classical approach, creating a new approach that we have named “Deep Sensing”.

Deep Learning and Neural Networks elaborate data and processes for a specific objective, so they dedicate a single algorithm to a precise purpose. Deep Sensing, instead, simulates the cognitive processes, not the structure of each sentence. So, it is able to adapt to any situation, assuming non automatic behaviors. For example, if you want an AI built with Deep Learning to play Poker, Blackjack and Chess, you must create an algorithm for each game because the process (NN) must be created in function of the input patterns.

Thanks to Deep Sensing, HIO has only a single algorithm able to play all of them and, more in general, to understand the process by which is possible to reach any target, because, in this case, it’s the process that adapts itself to the input patterns.

HIO is able to acquire experience, data and knowledge in all languages without manual configurations, manage unexpected behaviors and to simulate humans deduction using its own experience.

HIO's ease-of-use with the accuracy of its analysis, the ability to contextualize the information learned, represent the most appreciated key values by our customers.

UNXFLOW

An innovative logical entity, based on contexts and notions, which allows HIO to create a conversational workflow, not predefined, in runtime mode. UNXFLOW produces a conversational flow on a certain argument, in runtime mode. User might answer or answer with a new question. You can also change argument (or UNXFLOW) and resume a previous UNXFLOW.

During a conversation, or data analysis, HIO can run one or more UNXFLOW autonomously, transferring the input data from one UNXFLOW to other in function of the events. Thanks to UNXFLOW, HIO is able to manage autonomously more contemporary conversations keeping in memory all steps of the procedures not executed.

Artificial INTUITION Algorithm

HIO intuition is a process designed by studying the Human Intuition Processes able to identify a response, in function of an external stimulus, using logic and intuition, transversally across all acquired notions (Transfer Learning): HIO is able to apply the heuristic process, analyzing multi-disciplinary information independently of the context and of the language, without manual configurations.

HIO can have deep cognition of the questions asked by the user and of provided answers.
HIO can create concepts, identify new topics and subjects type, on the basis of logical deductions, simulating the humans learning logic and analytical abilities.

All response provided by HIO through the Artificial Intuition Algorithm, are the result of a deep reasoning across all acquired notions.

EXPERIENCE

Experience can’t be defined incrementally, because each new concept, word, subject, can modify, also radically, the previous knowledge.So, humans knowledge, change constantly in function of external stimuli. Simplifying: Knowledge is composed by experience and notions.

Having experience it’s very different to have only notions, because experience contains the practical application of the notions and processes by which the notions are applied. Notions acquires sense in function of a real problem to solve in a specific context.

To do good, it is first necessary to know it: There is no moral or ethics without the knowledge of what is morally right or wrong. Ethics and Emotion Intelligence aren’t processes or an innate dowry, but a set of notions, learned from external stimuli universally recognized as true and right, which have no purpose to increase knowledge, but rather to direct decision-making processes in practical application of functional knowledge.

WORKFLOWS

Humans can understand and learn fixed procedures by using all their own experience when it’s necessary, especially when something happens that the procedure had not foreseen.

Humans can also resolve an unexpected problem, maintaining in memory all information acquired applying the procedure, and to resume the previous suspended steps without starting over.

INTUITION

We have understood that Logical Deduction, Intuition and, maybe, Imagination, are the result of a different application of the same process. Results change in function of the origin of the information to which the process draws.

We have also understood that during the intuition process, our mind uses and analyzes all notions and experiences even if minimally related to the contexts of the problem to solve. During the decisional process, the meaning of the problem loses its own importance, and only to the end of processes, it acquires its meaning again, discarding all inconsistent relationships and coming to a conclusion.

In the Human Mind the “brilliant idea” happens because the Rational Deduction process draws on information memorized in the unconscious part of the memory. This may suggest that the idea was born for a randomness, in reality it’s a logical deductive process triggered by a need.

MIND VS BRAIN

Often, AI scientists try to explain the human behavior studying the neural interaction. For a Neuroscientist, it’s the right way. For a Psychologist / Psychoanalyst / Analogists / Cognitivists it’s a wrong way, because they study the causes-effects of human’s cognitive processes.

In our opinion, the AI growth has been limited because they try to explain process behaviors, studying the structure that, thanks to it, the process can run. To draw a parallel with informatics: It is like someone trying to understand a Mobile APP functionality by studying how smartphone’s hardware works. Maybe it isn’t a wrong way, but only a very slow one.
